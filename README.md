# ⚔️ Bleach API

API de personagens do anime Bleach feita com Node.js + Express.

## 🚀 Rotas

- GET / → mensagem inicial
- GET /bleach → todos os personagens
- GET /bleach/:id → busca por ID
- GET /bleach/nome/:nome → busca por nome
- GET /bleach/raca/:raca → busca por raça
